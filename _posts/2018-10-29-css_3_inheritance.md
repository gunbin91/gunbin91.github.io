# 상속 (inheritance )
css에는 하위태그 까지 상속되는 속성들이 있다. 상속을 사용하는 이유는 <font color="deeppink">생산성"</font>때문이다.<br/> 
( 최소한의 코드로 많은 부분을 적용할 수 있기 때문에 )

- 상속된 속성은 웹브라우저 개발자 모드에서 확인할 수 있다.
